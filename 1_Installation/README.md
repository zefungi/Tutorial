# Installation for Ubuntu

###### tags: `ubuntu`

### Step 1. 洗一個開機碟
Need:
1. 一顆空的隨身碟(8GB 比較保險)
3. 一個有 windows 的電腦
4. 開機碟製作軟體：[rufus](https://rufus.ie/en/)
5. [Ubuntu 對應版本的映像檔(.iso)](https://www.ubuntu-tw.org/modules/tinyd0/)

### Step 2. 劃分硬碟空間
手動切硬碟
- **efi**: 128 MB 即可
- **swap**: 看裝置的記憶體大小決定切多大
- **root**: 系統相關資料存放位置，不要設太小 (50G 應該夠用)
- **home**: 看硬碟還剩多少空間，剩下就都給他
